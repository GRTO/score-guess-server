## Description

Project created by [Nest](https://github.com/nestjs/nest). It is used to match numbers randomly generated. Here is a personal project for testing purposes of websockets.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
